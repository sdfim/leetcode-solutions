# Two Pointers Problems

This directory contains solutions to various "Two Pointers" problems categorized by difficulty and topic. Each problem includes a Python implementation.

## Structure

- **BASIC/**: Fundamental problems to get started with the two-pointers technique.
- **MEDIUM/**: Intermediate problems that require a deeper understanding of pointer manipulation, often combined with sorting or greedy approaches.
- **ADVANCED/**: Challenging problems that involve complex conditions, sliding windows with auxiliary data structures, or multi-stage algorithms.

## Topics Covered

- Classic Two Pointers (Left/Right)
- Fast and Slow Pointers (Cycle Detection)
- Sliding Window
- Partitioning (Dutch National Flag)
- Merge Operations
- Palindrome Verification
- N-Sum Problems (2Sum, 3Sum, 4Sum)
