# Linked List Problems

This directory contains solutions to various linked list problems categorized by difficulty and topic. Each problem includes a Python implementation and a detailed explanation.

## Structure

- **BASIC/**: Fundamental problems to get started with linked lists.
- **MEDIUM/**: Intermediate problems that require a deeper understanding of linked list operations.
- **ADVANCED/**: Challenging problems that involve complex algorithms and data structures.

## Topics Covered

- Traversal
- Insertion and Deletion
- Cycle Detection
- Reversal
- Merging
- Splitting
- Advanced Data Structures (e.g., Skiplist, LRU Cache)
